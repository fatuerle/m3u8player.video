What Is an M3U8 File?

An M3U8 file is a plain-text playlist format used by HTTP Live Streaming (HLS), the streaming protocol developed by Apple. Instead of delivering one giant video file, HLS splits the video into small segments and lists them in an M3U8 playlist.

M3U vs M3U8 - What's the Difference?

The original .m3u format uses plain ASCII encoding. The .m3u8 extension indicates the file uses UTF-8 encoding, which supports international characters and is required by the HLS standard.

Frequently Asked Questions

Why is my M3U8 stream not playing?
The most common cause is a CORS (Cross-Origin Resource Sharing) restriction on the source server. Web browsers block requests to a different domain unless the server gives permission. Try the CORS browser extension linked in our warning message.
Is this M3U8 player free to use?
Yes, M3U8Player.video is completely free. No sign-up, registration, or subscription is required to test and play your HLS streams online.
Does it support mobile devices and iOS?
Absolutely. Our player is mobile-friendly and supports native HLS playback on iOS Safari and modern Android browsers. Play your streams on the go without installing apps.
How to download an M3U8 stream as MP4?
Since M3U8 files are only playlists, you cannot "convert" them directly. You need a tool like FFmpeg or VLC to download all the video segments listed in the playlist and merge them into a single MP4 file.
Can I play M3U8 links on VLC Media Player?
Yes. Open VLC, go to Media > Open Network Stream, and paste your .m3u8 URL. VLC is excellent for playing HLS streams if you prefer a desktop application over a web player.
What is the difference between HLS and DASH?
HLS (HTTP Live Streaming) uses the M3U8 format and is widely supported by Apple. DASH (MPEG-DASH) uses XML-based .mpd manifests and is often used for DRM-protected content. Both use adaptive bitrate streaming to ensure smooth playback.
Why does my video keep buffering or lagging?
Buffering occurs when your internet speed is slower than the video's bitrate, or when the source server is overloaded. Adaptive bitrate HLS streams should automatically switch to a lower quality to prevent this.
Can I use this for IPTV M3U playlists?
This tool is designed for individual M3U8 HLS stream links. For massive IPTV playlists containing hundreds of channels, we recommend specialized IPTV software, though you can test individual channel links here.
Are M3U8 links safe to open?
M3U8 files are plain text documents and do not contain executable code. However, you should only play links from trusted sources to avoid tracking or exposure to inappropriate content.
Do I need to install any plugins?
No. This player uses HLS.js, a JavaScript library that works natively in any modern Chrome, Firefox, or Edge browser without requiring Flash or any other plugins.
